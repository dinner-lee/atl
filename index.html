Pass 입력
Fail 입력
모델 상태

학생별 토론 로그와 교사 판단

더미 CSV 지표를 보고 각 학생의 Pass/Fail을 직접 지정합니다. 기본값은 예시 판단으로 채워져 있습니다.

학습된 원 가중치 식

교사가 입력한 Pass/Fail 판단에 가장 가까워지도록 학습한 로지스틱 회귀식입니다.

정확도
절편

변수별 원 가중치

값은 표준화된 입력 기준입니다. 양수는 Pass 가능성을 높이고, 음수는 낮춥니다.

모델 예측 결과

교사 판단과 모델 예측을 비교합니다.

학생 교사 판단 모델 확률 모델 예측 일치

양의 상관 변수

교사 Pass 판단과 함께 증가한 변수입니다. 강한 변수부터 정렬됩니다.

음의 상관 변수

교사 Pass 판단과 반대로 움직인 변수입니다. 강한 변수부터 정렬됩니다.

가중치 조정 슬라이더

상관이 강한 변수 순서로 제시됩니다. 조정 즉시 학생별 Pass/Fail 예측이 바뀝니다.

조정 후 학생 판정

현재 슬라이더 값으로 다시 계산한 결과입니다. 확률 0.5 이상이면 Pass입니다.
