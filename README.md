# xtl: AI-Adaptive Teacher Dashboard (forked)

교사가 학급별 차시 데이터에 대해 학생 수준(상, 중, 하)을 직접 판단하면, 그 판단에 부합하는 로지스틱 회귀 모델을 실시간 학습하고 가중치 슬라이더를 미세조정하며 결과를 시뮬레이션할 수 있는 지능형 교사 대시보드입니다. 교사 선호도(좋아요/싫어요)에 반응하는 맞춤형 인사이트 뷰와 다양한 에듀테크 연동 기능을 제공합니다.

## 실행

- `data.js`와 `app.js`가 정상적으로 포함되어 있는지 확인하고, 브라우저에서 `index.html`을 열면 즉시 구동됩니다.
- 상태값(교사 판단 여부, To-Do 리스트 진행도, 시각화 선호도 등)은 브라우저의 `localStorage`에 자동 보관됩니다.

## 파이프라인 및 핵심 기능

1. **학급 및 차시별 적응형 관리**:
   - 교사의 최종 수동 판단 전, AI가 학생 참여도 데이터를 분석하여 캡슐 형태의 `잠정 진단 완료` 상태 표시
   - 각 차시별 교사 특이사항(To-Do 형태의 마크다운 리스트) 개별 편집 및 실시간 반영

2. **3단계 적응적 예측 모델 학습**:
   - **교사 판단 입력**: 5대 학습 피처 데이터를 참고하여 학생별 성취 수준(상, 중, 하)을 설정합니다.
   - **원 가중치 식**: 입력된 데이터를 기반으로 실시간 로지스틱 회귀 학습을 진행합니다.
   - **가중치 조절**: 각 변수의 가중치를 슬라이더로 직접 조절하여 예측 모델을 실시간으로 커스텀하고, 학생들의 판정 확률 변화 시뮬레이션을 관측합니다.

3. **교사 맞춤형 피드백 인사이트**:
   - 각 시각화 모듈을 누르면 **AI 맞춤형 진단 코멘트** 및 **수업 설계 제언(Action Plan)**을 담은 팝업창이 출력됩니다.
   - **좋아요/싫어요 선호도 반응형 뷰**: 교사가 메인 화면에서 싫어요를 누른 시각화는 숨겨집니다. 단, 우측 상단 **'전체 분석 리포트'** 버튼을 누르면 선호도 필터를 우회하여 모든 시각화 모듈이 누락 없이 우선순위 순으로 정렬되어 복구 출력됩니다.
