# aTL: Adaptive Teaching and Learning

교사가 학생별 Pass/Fail을 직접 입력하면, 그 판단에 가까운 로지스틱 회귀 가중치를 학습하고, 이후 교사가 슬라이더로 변수 가중치를 조정하며 학생별 Pass/Fail 변화를 관찰할 수 있는 적응적 대시보드입니다.

## 실행

파일 전체(더미 csv 파일 제외)를 다운로드하고, 브라우저에서 `index.html`을 열면 사용할 수 있습니다.

## 파이프라인(?)

1. 20명 학생의 토론 더미 로그를 확인하고 교사 Pass/Fail을 입력합니다.
2. 로지스틱 회귀로 교사 판단에 가까운 가중치 식을 학습합니다.
3. 상관관계가 강한 변수 순서로 슬라이더를 조정하며 직접 Pass/Fail 가중치 식을 수정합니다.

## 더미 변수

더미 변수는 다음과 같습니다. 변수의 수가 달라지거나 xAPI 등을 쓰게 되는 경우에는 일부 수정이 필요할 수 있습니다.
- 게시글 작성 횟수
- 댓글 작성 횟수
- 자신의 게시글 좋아요 수
- 누른 좋아요 수
- 자신의 댓글 좋아요 수
- 질문 작성 횟수
- 동료에게 단 댓글 수
- 근거/자료 언급 횟수

## 추후 개선 사안

추후 개선할 여지가 있는 부분은 다음과 같습니다. 작성 순서는 개발 순서와 0.5 이하의 상관 관계를 가지고 있습니다.

- 여러 회차의 수업을 누적하여 데이터를 제공할 수 있는 기능
- 누적된 데이터와 교사 판단을 이용한 강화 학습으로 여러 예상 피드백 및 점수 식을 제안하는 기능
- 표 이외에도 여러 형태의 시각화된 데이터를 제공하는 기능
