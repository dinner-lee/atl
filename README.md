# Teacher-Calibrated Discussion Dashboard

교사가 학생별 Pass/Fail을 직접 입력하면, 그 판단에 가까운 로지스틱 회귀 가중치를 학습하고, 이후 교사가 슬라이더로 변수 가중치를 조정하며 학생별 판정 변화를 확인하는 정적 프로토타입입니다.

## 실행

브라우저에서 `index.html`을 열면 됩니다. 별도 서버나 설치는 필요하지 않습니다.

## 흐름

1. 20명 학생의 토론 더미 로그를 확인하고 교사 Pass/Fail을 입력합니다.
2. 로지스틱 회귀로 교사 판단에 가까운 원 가중치 식을 학습합니다.
3. 상관관계가 강한 변수 순서로 슬라이더를 조정하며 Pass/Fail 변화가 즉시 반영되는지 봅니다.

## 더미 변수

- 게시글 작성 횟수
- 댓글 작성 횟수
- 자신의 게시글 좋아요 수
- 누른 좋아요 수
- 자신의 댓글 좋아요 수
- 질문 작성 횟수
- 동료에게 단 댓글 수
- 근거/자료 언급 횟수
